# no-lose-janken-ai

## 前提条件

- **OS**: Windows 11  
- **CUDAバージョン**: 12.4  

---

## 使用方法

1. **仮想環境の作成**  
   以下のコマンドで仮想環境を作成します。  

   ```bash
   python -m venv venv
   venv\Scripts\activate.bat
   pip install -r requirements.txt
   ```
2. **アプリの起動**  
   以下のコマンドでアプリを実行します。
   ```bash
   python -m backend.main
   ```
